# WEEKLY REVIEW CHECKLIST

**Use this checklist every Friday 3-4 PM (Solo Mode) or Sunday 8:30-9:30 PM (Family Mode).**

Print this page or check boxes directly in your browser (progress saves automatically).

{% capture section_1 %}
### Google Keep Chaos Audit (10 minutes)
- [ ] Open Google Keep and review all notes from this week
- [ ] Process actionable tasks → Move to Google Tasks
- [ ] Process project ideas → Move to NotebookLM with tag "Ideas - [Module]"
- [ ] Process reference info → Move to NotebookLM with tag "Reference - [Topic]"
- [ ] Archive already-handled notes
- [ ] Delete no-longer-relevant notes
- [ ] Tag unclear items "Review Next Week"
- [ ] **Goal**: Google Keep is empty (or only has "Review Next Week" items)

### Voice Recording Processing (5 minutes)
- [ ] Open Soundcore AI Recorder app
- [ ] Review untranscribed recordings
- [ ] Extract key information from each
- [ ] File in Google Keep / Tasks / NotebookLM
- [ ] Delete recordings after filing
{% endcapture %}

{% capture section_2 %}
### Completed Task Audit (5 minutes)
- [ ] Open Google Tasks and review all "Completed" tasks
- [ ] Note which tasks took more/less time than expected
- [ ] Note what made tasks easy or hard
- [ ] Identify recurring tasks that should be scheduled
- [ ] Track productivity patterns (which days, which types)

### Incomplete Task Triage (5 minutes)
- [ ] Review all "Not Completed" tasks
- [ ] For each: Is it still relevant? (Yes = keep, No = delete)
- [ ] For each: Why didn't I complete it? (vague, too big, wrong priority, blocker)
- [ ] Rewrite vague tasks with clear micro-actions
- [ ] Break big tasks into smaller tasks
- [ ] Archive or delete non-relevant tasks
- [ ] Update deadlines for carryover tasks
{% endcapture %}

{% capture section_3 %}
### Foundational Protocol Compliance (5 minutes)
Rate each protocol (7/7 = Excellent, 5-6/7 = Good, 3-4/7 = Needs Attention, 0-2/7 = Critical)

- [ ] **Water Protocol**: Did I drink water immediately upon waking every day?
- [ ] **Pill Protocol**: Did I take pills every morning?
- [ ] **Capture Protocol**: Did I capture fleeting thoughts throughout the week?
- [ ] **Sleep Anchor**: Did I go to bed at 10:30 PM consistently?
- [ ] **Daily Dump**: Did I review tomorrow's tasks before bed each night?
- [ ] For "Needs Attention" or "Critical Failure": Identify barrier and plan fix

### Mode-Specific Protocol Review (3 minutes)
- [ ] **If Solo Mode**: Protected core work blocks (9-1, 2-5)?
- [ ] **If Solo Mode**: Deployed Bubble Shield Profile 1.0 during focus time?
- [ ] **If Solo Mode**: Maintained meal prep and exercise protocols?
- [ ] **If Family Mode**: Enforced Quiet Time Protocol (1-3 PM)?
- [ ] **If Family Mode**: Used Bubble Shield Profile 2.0 appropriately?
- [ ] **If Family Mode**: Maintained structured family schedules?
- [ ] Identify what broke down and why

### IPI Usage Tracking (2 minutes)
- [ ] Count: How many times did I use Executive Dysfunction Emergency Protocol?
- [ ] Count: How many times did I use Context Switching Recovery Protocol?
- [ ] Count: How many times did I use Sensory Overload Emergency Protocol?
- [ ] For each: What triggered it?
- [ ] For each: Did it work?
- [ ] For each: Can I prevent the need next week?
- [ ] **Red Flag**: If any IPI used 3+ times, root cause analysis needed
{% endcapture %}

{% capture section_4 %}
### Calendar Review (5 minutes)
- [ ] Open calendar and review next week (Monday-Sunday)
- [ ] For each event: Is this still necessary?
- [ ] For each event: What preparation does it require?
- [ ] For each event: What is the sensory/cognitive cost?
- [ ] For each event: Does this conflict with my mode schedule?
- [ ] Add missing events (meal prep, Quiet Time, exercise, next Weekly Review)

### Priority Task Selection (5 minutes)
- [ ] From cleaned task list, identify 3-5 Priority Tasks for next week
- [ ] Priority criteria: Has deadline / Moves project forward / Postponed multiple weeks / High impact
- [ ] Assign each Priority Task to specific day
- [ ] Estimate time required for each
- [ ] Block time on calendar for completion

### Risk Assessment (3 minutes)
- [ ] Identify schedule risks (back-to-back meetings, overcommitted days)
- [ ] Identify sensory risks (high-stimulation events, long social days, travel)
- [ ] Identify transition risks (mode switches mid-task, deadline clustering)
- [ ] Identify health risks (poor sleep expected, medication low, illness)
- [ ] For each risk: Create mitigation plan

### Success Definition (2 minutes)
- [ ] Write: "Next week will be successful if I accomplish these 3 things:"
- [ ] 1. [Priority Task 1]
- [ ] 2. [Priority Task 2]
- [ ] 3. [Maintain foundational protocols]
{% endcapture %}
