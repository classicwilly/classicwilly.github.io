# Wonky Sprout Operating System (WS-OS)

## Overview
The **Wonky Sprout Operating System (WS-OS)** is a neurodivergent life management system designed to provide structured protocols for individuals managing Autism and ADHD. Built using Jekyll and hosted on GitHub Pages, this project aims to create an "anti-BS" environment that transforms chaos into order.

## Project Structure
- **_config.yml**: Configuration settings for the Jekyll site.
- **index.md**: Main landing page with an introduction and links.
- **catalog.md**: Index of available content on the site.
- **sop-vault.md**: Public index of all Standard Operating Procedures (SOPs).
- **_posts/**: Contains blog posts in Jekyll format.
- **_layouts/**: Default layout for the site.
- **_includes/**: Header and footer sections for reuse across pages.
- **_sass/**: SASS variables for consistent theming.
- **_data/**: Navigation data for menus and links.
- **pages/**: Content pages describing various functionalities.
- **assets/**: Contains CSS, JavaScript, and libraries for styling and functionality.
- **scripts/**: Automation scripts for building the visual engine.
- **.github/**: GitHub Actions workflows for deployment.
- **package.json**: npm configuration file for dependencies.
- **tsconfig.json**: TypeScript configuration file.

## Getting Started
1. Clone the repository:
   ```
   git clone https://github.com/classicwilly/wonky-sprout-os.git
   ```
2. Navigate to the project directory:
   ```
   cd wonky-sprout-os
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Build the project:
   ```
   npm run build
   ```
5. Serve the site locally:
   ```
   npm run serve
   ```
