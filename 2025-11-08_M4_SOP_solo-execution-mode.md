<h2 id="daily">DAILY SCHEDULE (HOURLY BREAKDOWN)</h2>

### 6:00 AM - Morning Protocols
- **FDP Active**: Water Protocol ‚Üí Pill Protocol (3 minutes)
- **Status**: Foundation running

### 6:03 AM - 7:00 AM | Morning Routine (57 minutes)
- **Shower & Hygiene** (15 min)
- **Get Dressed** (5 min)
- **Breakfast** (15 min) - Pre-portioned meal from meal prep
- **Command Center Setup** (10 min):
  - Boot VS Code
  - Open NotebookLM
  - Check Google Tasks for 3 critical tasks
  - Engage Bubble Shield Profile 1.0 (Transparency 3, Vol 20%)
- **Buffer** (12 min) - Final prep, mental transition

### 7:00 AM - 8:00 AM | Planning & Filing (1 hour)
- **Protocol**: The Card Catalog Time
- **Action**:
  1. Open Google Keep ‚Üí Process all captures from yesterday
  2. Open Soundcore AI Recorder ‚Üí Review all voice captures
  3. File critical information:
     - Important data ‚Üí NotebookLM
     - Code/system notes ‚Üí VS Code
     - Tasks ‚Üí Google Tasks
  4. Load today's 3 critical tasks into working memory
  5. Clear all temporary storage (Keep notes archived)
- **Tool**: NotebookLM, VS Code, Google Keep, Soundcore Recorder
- **Output**: Clean mental system, 3 tasks loaded and prioritized

### 8:00 AM - 12:00 PM | Core Work Block 1 (4 hours)
- **Protocol**: Deep Work Block
- **Action**:
  - Work ONLY on Task 1 (highest priority)
  - Pomodoro Technique: 25 min focus / 5 min rest
  - 8 Pomodoros total (4 hours)
  - Bubble Shield Profile 1.0 ENGAGED (no deviation)
- **Tool**: Soundcore Space One headphones, Pomodoro timer
- **Environment**: 
  - Phone on DND
  - VS Code full screen
  - No browser tabs except necessary docs
- **Execution Standard**: Reference `2025-11-08_M4_SOP_classicwilly-protocol.md` for 11/10 output

### 12:00 PM - 12:30 PM | Lunch & Reset (30 minutes)
- **Eat Pre-Portioned Meal** (15 min)
- **Tidy Block** (15 min):
  - Reset Command Center (clear desk, organize cables)
  - Quick kitchen cleanup
  - Everything returns to designated place

### 12:30 PM - 3:30 PM | Core Work Block 2 (3 hours)
- **Protocol**: Deep Work Block (continued)
- **Action**:
  - Work on Task 2 (if Task 1 complete) OR continue Task 1
  - Pomodoro Technique: 25 min focus / 5 min rest
  - 6 Pomodoros total (3 hours)
  - Bubble Shield Profile 1.0 MAINTAINED
- **Tool**: Same as Block 1
- **Note**: This is the final push for critical work

### 3:30 PM - 4:00 PM | Movement Protocol (30 minutes)
- **Protocol**: Regulation Fix
- **Action**: Choose ONE:
  - Weightlifting session (garage gym)
  - 30-minute walk (neighborhood loop)
  - Dedicated gardening (physical labor)
- **Purpose**: Dopamine regulation, impulse control reset
- **Non-Negotiable**: Must be physical exertion

### 4:00 PM - 5:00 PM | Buffer & Admin (1 hour)
- **Low-Priority Tasks**:
  - Respond to emails (if needed)
  - System maintenance (updates, backups)
  - Light planning for tomorrow
  - Review Google Tasks progress
- **Note**: This is NOT deep work time‚Äîlight cognitive load only

### 5:00 PM - 6:00 PM | Dinner & Wind Down (1 hour)
- **Eat Pre-Portioned Meal** (20 min)
- **Tidy Block** (15 min):
  - Full Command Center reset
  - Kitchen cleanup
  - Prepare environment for next day
- **Free Time** (25 min):
  - Reading
  - Light hobby work
  - Low-stimulation activity

### 6:00 PM - 10:00 PM | Personal Time (4 hours)
- **Unstructured Time** (within reason):
  - Personal projects (3D printing, CAD, electronics)
  - Entertainment (reading, TV, music)
  - Social (if applicable)
- **Capture Protocol**: STILL ACTIVE (capture any work thoughts)
- **Note**: No high-stakes work‚Äîthis is regulation/recovery time
